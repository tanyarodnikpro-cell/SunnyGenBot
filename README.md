# SunnyGenBot
Уютный Telegram-бот для выгорающих офисных людей ☀️

## Приватность

- Бот не сохраняет Telegram ID, имена, username, переписку или выбранный режим на диск.
- Выбранный режим и согласие на AI-обработку существуют только в оперативной памяти и исчезают после перезапуска.
- Рассылки и постоянный список подписчиков отключены.
- Текст передаётся в OpenAI только после явного согласия пользователя.
- Согласие можно отозвать командой `/revoke`.
